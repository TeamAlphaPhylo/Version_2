Virtual Phylo README

============================================

Virtual Phylo is an iPad application for playing
the collectable card game Phylo. This application
emulates the aspects of playing the game, building
decks and collecting new cards.

for details of Phylo see http://phylogame.org/

Virtual Phylo is found on Github at 
https://github.com/TeamAlphaPhylo/Tutorial

Virtual Phylo Requirements:
Platform: Apple iPad 
OS version: 4.0+
